# Timeline

![GitHub](https://img.shields.io/github/license/gcnyin/timeline)

A web application.

- scala
- playframework
- slick
- flyway
- postgresql
- bootstrap

## Build

### Docker

```
sbt playUpdateSecret docker:publishLocal
```

## Run

```
docker-compose up -d
```

Service is listening on port `9000`.
